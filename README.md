# NewsArea

NewsArea is a responsive React news application built with Vite, NewsAPI, Material UI, Bootstrap, and infinite scrolling.

## Features

- Top headlines by category
- Infinite scroll pagination
- Responsive desktop/mobile navigation
- Dark mode toggle
- Skeleton loading cards
- Loading progress indicator
- Graceful fallback UI for missing images and API errors

## Tech Stack

- React
- Vite
- Material UI
- Bootstrap
- React Router DOM
- NewsAPI

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file from the example:


Then add this inside .env:
VITE_NEWS_API=your_api_key_here
